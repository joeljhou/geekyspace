# 依赖（Dependencies）

企业应用程序通常不是由单个对象（或`Spring`术语中的`Bean`）构成的。
即使是最简单的应用程序，也是由多个对象共同协作来呈现给最终用户一个连贯的应用体验。

下一节将解释如何从定义独立的Bean开始，逐步实现一个完整的应用程序。
在这个应用程序中，各个对象将相互协作，实现一个共同的目标。
