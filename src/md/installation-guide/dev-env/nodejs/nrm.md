# npm源切换加速利器—nrm

主要官方来源：[npmjs-nrm](https://www.npmjs.com/package/nrm)

> `nrm`可以帮助您轻松快速地切换不同的 npm 注册表。它支持 `npm` 、 `cnpm` 、 `taobao` 、 `yarn` 、 `tencent` 、 `npmMirror`和`huawei` 。

## 安装nrm
```shell
# npm 安装
npm install -g nrm
```
安装成功后，使用`nrm --version`查看命令，验证是否安装正常。
